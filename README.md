# Content Investment ROI Analysis

## Objective
Evaluate whether higher production budgets lead to stronger financial returns and inform greenlight decisions.

## Approach
- Simulated production budgets in Python to enable ROI analysis
- Cleaned and structured dataset (budget, revenue)
- Created ROI metric: (Revenue - Budget) / Budget
- Analyzed relationship between budget size and return
- Visualized top-performing content and ROI distribution

## Key Insights
- Mid-budget content delivers more consistent ROI
- High-budget projects show higher variance and risk
- Outliers skew perceived success of large investments

## Tools
Python (pandas, matplotlib), Google Colab

## Business Impact
Replicates real-world studio greenlight and portfolio strategy decision frameworks.
